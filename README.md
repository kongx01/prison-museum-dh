# prison-museum-dh
Data and code for master's thesis on prison museum dark tourism in East Asia
# 동아시아 다크투어리즘 공간 스토리텔링 연구 
본 저장소는 KONG XUAN의 석사학위논문 「동아시아 다크투어리즘 공간 스토리텔링 연구 ― 한중일 감옥 박물관을 중심으로 ―」(고려대학교대학원, 2026)에 사용된 웹 크롤링 코드 및 수집 데이터를 공개한다.

## 저장소 구성

- `naver/`: 네이버 블로그(서대문형무소역사관 관련) 수집 코드
- `jalan/`: 자란(아바시리감옥박물관 관련) 수집 코드
- `data/`: 세 박물관의 수집 원본 데이터

## 수집 데이터 개요

| 박물관 | 플랫폼 | 수집 건수 | 수집 기준일 |
|---|---|---|---|
| 뤼순일러감옥구지박물관 | 샤오훙슈(小红书) | 3,554편 | 2026년 4월 22일 |
| 서대문형무소역사관 | 네이버 블로그 | 998편 | 2026년 5월 2일 |
| 아바시리감옥박물관 | 자란넷(じゃらんnet) | 1,471편 | 2026년 5월 6일 |

## 샤오훙슈(小红书) 데이터에 대하여

샤오훙슈 데이터의 경우, 플랫폼 정책상 수집 코드는 공개되지 않는다. 수집된 원본 데이터는 `data/` 디렉터리에서 확인할 수 있다.

## 사용 환경

- Python 3.10 이상
- 각 디렉터리의 `requirements.txt` 참조
