# Data

본 연구에서 수집된 원본 데이터.

## 파일 구성

- `xiaohongshu_raw.xlsx`: 샤오훙슈에서 '旅顺日俄监狱旧址博物馆' 키워드로 수집된 노트 (3,554편)
- `naver_blog_raw.xlsx`: 네이버 블로그에서 '서대문형무소역사관' 키워드로 수집된 포스트 (998편)
- `jalan_kuchikomi_raw.xlsx`: 자란에서 아바시리감옥박물관에 대해 수집된 일본인 후기 (1,471편)

## 수집 일자

- 샤오훙슈: 2026년 4월 22일까지의 공개 노트
- 네이버 블로그: 2026년 5월 2일 수집
- 자란넷: 2026년 5월 6일까지 등록된 후기

## 수집 방법

- 샤오훙슈: 플랫폼 정책상 수집 코드는 비공개
- 네이버 블로그: `naver/` 디렉터리의 스크립트로 수집
- 자란넷: `jalan/` 디렉터리의 스크립트로 수집

## 파일 형식 안내

데이터는 Excel(.xlsx) 형식으로 제공된다. Microsoft Excel, 
LibreOffice, Numbers 등의 일반 표 계산 소프트웨어로 열람할 수 있다.
